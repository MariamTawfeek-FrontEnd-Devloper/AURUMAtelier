# Aurum Atelier 💍

Welcome to the Aurum Atelier portfolio project! This is a modern, responsive web interface designed to showcase luxury jewelry collections with a focus on seamless user experience.

## 🚀 Features
- **Dynamic Product Rendering:** Products are injected dynamically using JavaScript for better maintainability.
- **Interactive Cart System:** Users can add items to their cart with real-time UI updates.
- **Smart UI Management:** The cart count badge intelligently appears only when items are added, utilizing Tailwind CSS for sleek styling.
- **Clean Architecture:** Refactored for professional standards, avoiding inline events and ensuring efficient DOM manipulation.

## 🛠 Tech Stack
- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **Tailwind CSS**
